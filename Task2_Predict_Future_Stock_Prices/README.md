# Task 2: Stock Price Prediction

# Objective
Predict the next day's closing stock price using historical data.

# Dataset
Stock data retrieved using yfinance (Apple stock)

# Models Applied
- Linear Regression
- Random Forest Regressor

# Steps Performed
- Data collection using yfinance
- Data preprocessing and feature selection
- Model training and testing
- Prediction and model evaluation

# Key Results
- Linear Regression performed better than Random Forest
- Predictions closely follow actual stock trends

## Tools Used
Python, Pandas, yfinance, Scikit-learn, Matplotlib
