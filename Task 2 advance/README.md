Task 2 – End-to-End ML Pipeline for Customer Churn Prediction
Objective Build a reusable, production-ready machine learning pipeline using Scikit-learn's Pipeline API to predict customer churn on the Telco dataset.
Dataset Telco Customer Churn Dataset – IBM sample dataset loaded directly from GitHub via URL. Contains ~7,000 customer records with 20 features including contract type, monthly charges, tenure, and internet service details.
Methodology / Approach 1. Loaded and cleaned the Telco dataset — converted TotalCharges to numeric, dropped null rows and irrelevant columns 2. Split features into numerical and categorical groups 3. Built separate sub-pipelines for each: Numerical: Median imputation → StandardScaler Categorical: Most-frequent imputation → OneHotEncoder 4. Combined using ColumnTransformer as the shared preprocessor 5. Trained two full pipelines: Logistic Regression – baseline model Random Forest – with GridSearchCV over n_estimators, max_depth, and min_samples_split 6. Evaluated both models using Accuracy, ROC-AUC, and Classification Report 7. Visualized confusion matrices and top 15 feature importances 8. Exported both pipelines using joblib and validated by reloading and predicting
